Online lookup

Look up the word or phrase after you press Enter or use selection lookup. No API key is needed.

Only an actively queried term is sent to an online provider. Local glossaries and webpage content stay on this device.

Sentence translation Experimental

When the query contains two or more English words, DictFloat can run the enabled providers directly after you press Enter or click the selection marker.

Sentence translation providers only return Chinese translation. They do not expose prompt settings. API keys are stored separately and are not included in normal backup export.

Web bridge providers are experimental. They reuse normal translation pages and may fail if the page is not logged in, has a captcha, or changes its DOM.

AI Providers OpenAI-compatible

Use DeepSeek, Doubao / Volcengine Ark, OpenAI, OpenRouter, One API, LiteLLM, or your own compatible endpoint. Each provider has its own API, model, prompt, and switch.

0/0 enabled

Enabled AI providers are shown after the normal sentence translation providers. Only selected text is sent after you manually trigger lookup.

Default prompts are prefilled and editable. Once you edit a prompt or other field, DictFloat keeps using your modified version. API keys are stored separately in chrome.storage.local and are not included in normal backup export.
