DictFloat

Compact floating dictionary · Local-first

v0.3.7

Selection lookup

Choose what happens after text is selected on a webpage.

Floating window

Light uses an opaque white window, even when Chrome and the webpage are dark.

Online lookup

Look up the word or phrase after you press Enter or use selection lookup. No API key is needed.

Only an actively queried term is sent to an online provider. Local glossaries and webpage content stay on this device.

Local glossaries

Create separate PCIe, GPU, firmware, or learning glossaries. Disabled glossaries stay local but do not participate in search.

Lookup order

DictFloat shows matching dictionaries from top to bottom. Use ↑ and ↓ to choose what appears first. In the floating window, each dictionary result can be collapsed independently.

Wudao offline dictionary Optional

Import a local Wudao-style data pack. DictFloat keeps the four selected files in its own local IndexedDB and reads them offline.

Select en.ind, en.z, zh.ind, and zh.z together from a local copy you are permitted to use. The pack is not bundled with DictFloat.

MDX / MDD dictionaries Text lookup beta

DictFloat now indexes MDX text entries locally so they can participate in lookup order. MDD images and audio are detected but are not rendered in this text-first phase.

For multi-part resources, select the folder or select the main .mdx plus matching .mdd, .1.mdd, .2.mdd files together. JavaScript inside a dictionary folder is never executed. Import can take a while for large dictionaries because the text index is built locally.

Backup and import

JSON preserves glossary grouping, related terms, and all entry fields. CSV is suitable for Excel.
