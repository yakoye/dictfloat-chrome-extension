MDX / MDD dictionaries Beta

This first v0.3 step reads local MDX header metadata and records MDX / MDD pairs as independent sources. Full entry indexing and definition rendering are the next decoder step.

Select one .mdx file and any matching .mdd resource files together. Files are inspected locally; this build stores source metadata only, not a copied multi-GB dictionary file.
