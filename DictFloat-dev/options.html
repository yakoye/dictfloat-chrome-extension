DictFloat

Compact floating dictionary · Local-first

v0.2.0

Selection lookup

Choose how DictFloat responds when you select text on a webpage.

Floating window

The default is intentionally compact: 12px text, 380px width. “Follow Chrome” updates automatically when Chrome changes between light and dark mode.

Online lookup

Use public online dictionary and translation services when local entries do not cover the query. No API key is required in this version.

English queries can show an English definition and Chinese translation. Chinese queries can show an English translation. The selected text or typed query is sent only when you explicitly run a lookup.

My glossary

Import and export your local technical glossary. JSON preserves every field. CSV is suitable for Excel.

MDX / MDD dictionaries

Planned for v0.3. DictFloat v0.2 now supports online lookup plus a local glossary workflow; it does not yet parse proprietary MDX/MDD dictionary files.
