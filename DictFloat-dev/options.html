Selection lookup

Choose what happens after text is selected on a webpage.

Floating window

Light uses an opaque white window, even when Chrome and the webpage are dark.

Online lookup

Look up the word or phrase after you press Enter or use selection lookup. No API key is needed.

Only an actively queried term is sent to an online provider. Local glossaries and webpage content stay on this device.

Sentence translation Experimental

When the query contains two or more English words, DictFloat can run the enabled providers directly after you press Enter or click the selection marker.

Web bridge providers are experimental. They reuse normal translation pages and may fail if the page is not logged in, has a captcha, or changes its DOM.

Custom AI Provider OpenAI-compatible

Use DeepSeek, Doubao / Volcengine Ark, OpenAI, OpenRouter, One API, LiteLLM, or your own compatible endpoint. Only selected text is sent after you manually trigger lookup.

Use {{text}} where selected text should be inserted. Base URLs are automatically converted to /chat/completions.


      
API Key is stored separately in chrome.storage.local and is not included in normal backup export.
